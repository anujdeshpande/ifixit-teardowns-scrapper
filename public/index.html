ChipCensus

which products use which chips

ChipCensus is an index of the integrated circuits found inside consumer electronics. It answers two questions hardware folks actually ask: which devices use a given chip, and which chips show up again and again across the industry.

  • Source: staff-authored teardowns on iFixit (filtered for quality); ChipCensus is an independent project, not affiliated with iFixit.
  • Best-effort automated extraction — manufacturer attribution can be imperfect, and older teardowns list far more silicon than newer video-based ones.
